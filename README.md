# 🎮 Pokemulator

Bienvenue sur **Pokemulator** ! 
Pokemulator est un émulateur Game Boy, Game Boy Color et Game Boy Advance conçu pour être simple, élégant et prêt à jouer. Propulsé par le puissant moteur mGBA et doté d'une interface graphique moderne, il offre une expérience de jeu fluide et personnalisable.

---

## 📥 1. Installation

1. Téléchargez le fichier d'installation : `Install_Pokemulator_v1.0.exe`.
2. Double-cliquez sur le fichier pour lancer l'assistant d'installation.
3. Suivez les instructions à l'écran. Nous vous recommandons de cocher la case **"Créer un raccourci sur le bureau"**.
4. Une fois l'installation terminée, l'émulateur est prêt, mais il a besoin de jeux pour fonctionner !

---

## 🕹️ 2. Comment ajouter des jeux (ROMs)

*Note légale : Vous devez posséder les jeux originaux pour utiliser leurs fichiers ROM.*

Pokemulator lit les fichiers aux formats **.gba**, **.gbc**, et **.gb**.

1. Faites un **clic droit** sur le raccourci *Pokemulator* situé sur votre bureau.
2. Cliquez sur **"Ouvrir l'emplacement du fichier"**.
3. Dans le dossier qui s'ouvre, vous verrez un sous-dossier nommé **`roms`**. Ouvrez-le.
4. Glissez-déposez vos fichiers de jeu dans ce dossier.
   > **⚠️ Attention :** Si Windows vous affiche un message *"Vous devez disposer des droits d'administrateur"*, c'est tout à fait normal. Cliquez simplement sur **Continuer**.

---

## 🚀 3. Lancer le jeu et jouer

1. Lancez **Pokemulator** depuis votre bureau ou votre menu Démarrer.
2. L'interface principale affiche automatiquement tous les jeux présents dans votre dossier `roms`.
3. Cliquez sur le nom du jeu de votre choix pour lancer l'émulation.

### ⌨️ Contrôles par défaut (Clavier)
* **Direction :** Flèches du clavier
* **Bouton A :** Touche `X`
* **Bouton B :** Touche `C`
* **START :** Touche `Entrée`
* **SELECT :** Touche `Espace`
* **Gâchette L :** Touche `A`
* **Gâchette R :** Touche `E`

### ⚙️ Options en jeu
* **Mettre en pause :** Appuyez sur la touche `Échap (ESC)` à tout moment pour figer le jeu et couper le son.
* **Changer les touches :** Allez dans la barre de menu supérieure : *Configuration > Controles...*, puis cliquez sur le bouton "Modifier" de la touche que vous souhaitez changer.
* **Sauvegardes :** Sauvegardez normalement dans le jeu (ex: *Start > Sauvegarder*). L'émulateur créera automatiquement un fichier `.sav` dans le dossier `saves` à la fermeture de l'application.
